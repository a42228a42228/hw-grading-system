### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/a42228a42228/hw-grading-system.git
   ```
2. Setup environment 
   ```sh
   python ./hw-grading-system/setup.py
   ```
3. Add files into `data/`


## Usage

Run the programe
   ```sh
   python ./hw-grading-system/hw_grading_system/main.py
   ```
